Flexbox Grid TGAM

Flexbox Grid for TGAM

A grid system based on the flex display property.
With similiar functionality to it's predecessor and Bootstrap 4.
Supports different column counts for multiple tiers.

BEM Style Syntax

    </div>

Justify Content (horizontally)

Add classes to align elements to the start or end of a row as well as the top, bottom, or center of a column

Start
<div class="l-row u-justify-content-start">
  <div class="l-col-sm-4">
    <div class="box">
      Start
    </div>
  </div>
</div>
Center
<div class="l-row u-justify-content-center">
  <div class="l-col-sm-4">
    <div class="box">
      Center
    </div>
  </div>
</div>
End
<div class="row u-justify-content-end">
  <div class="l-col-sm-4">
    <div class="box">
      End
    </div>
  </div>
</div>

Example
