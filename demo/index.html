Flexbox Grid Sass

A grid system based on the flex display property.
With similiar functionality to it's predecessor and Bootstrap 4.
Supports different column counts for multiple tiers.

Download Zip Github Repository

Responsive

Responsive modifiers enable specifying different column sizes, offsets, alignment and distribution at xs, sm, md & lg viewport widths.

<div class="row">
  <div class="l-col-1
              l-col-sm-4
              l-col-md-8
              l-col-lg-13">
      <div class="box">Responsive</div>
  </div>
</div>

Fluid

Percent based widths allow fluid resizing of columns and rows.

Simple Syntax

All you need to remember is row, column, content.

<div class="row">
  <div class="l-col-xs-12">
    <div class="box">12</div>
  </div>
</div>

Offsets

Offset a column

<div class="row">
  <div class="l-col-xs-offset-3 l-col-xs-9">
    <div class="box">offset</div>
  </div>
</div>

Auto Width

Add any number of auto sizing columns to a row. Let the grid figure it out.

<div class="row">
  <div class="l-col-xs">
    <div class="box">auto</div>
  </div>
</div>

Nested Grids

Nest grids inside grids inside grids.

<div class="row">
  <div class="l-col-xs">
    <div class="box">auto</div>
      <div class="row">
        <div class="l-col-xs">
          <div class="box">auto</div>
        </div>
      </div>
    </div>
  </div>
</div>
